## x-login skill 修复

### 问题
- Python 版本需要 playwright 模块，但机器上没有 pip
- 之前的登录逻辑不符合 SKILL.md 描述的流程

### 解决方案
1. 用 Node.js 重写了 login.js（直接用 node_modules 里的 playwright）
2. 修复了登录流程：
   - 优先用保存的 cookies
   - 失败则自动用环境变量 AUTH_TOKEN/CT0 登录
   - 自动保存新 cookies

### Bug 修复
- 修复了 `context.cookies()` 在 browser 关闭后报错的问题
- 修复了登录检测逻辑（`https://x.com/` 被误判为登录成功）
- 修复了 cookies 加载方式（用 `storageState` 而非直接传 cookies）

### 最终可用的命令
```bash
node /home/node/.openclaw/skills/x-login/scripts/login.js        # 默认流程
node /home/node/.openclaw/skills/x-login/scripts/login.js --check  # 检查状态
node /home/node/.openclaw/skills/x-login/scripts/login.js --refresh # 强制刷新
```
