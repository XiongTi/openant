# 2026-02-20 工作日志

## 浏览器代理问题 - 总结

### 问题
- Chromium 需要 `--proxy-server` 参数才走代理
- 环境变量 `HTTP_PROXY` / `HTTPS_PROXY` 对 Chromium 不生效
- OpenClaw 浏览器配置 (`openclaw.json`) 目前不支持添加额外 chromium 参数

### 验证过程
- 手动测试确认：`/usr/bin/chromium --proxy-server=http://127.0.0.1:10809 --dump-dom https://www.baidu.com` ✅ 工作
- 直接运行 chromium 不加参数：❌ 不走代理

### 当前状态
- jina-reader 方案可用：用于抓取网页内容
- 浏览器代理：需要 OpenClaw 后续版本支持

### 后续
- 等待 OpenClaw 浏览器功能支持 proxy-server 参数
- 或联系开发者添加该功能
